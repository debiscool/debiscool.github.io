Imprimante.

Le système principal d'impression est le Common UNIX Printing System (CUPS) Le site ici

CUPS est le service d'impression par défaut.
CUPS détecte automatiquement les imprimantes.

Il peut être configuré et contrôlé en utilisant une interface web ou une interface graphique.

  • Utiliser une interface web : CUPS est accessible à "http://localhost:631/admin".
  • Utiliser une interface graphique : CUPS prend en charge une interface utilisateur graphique qui est
    "system-config-printer"

Pour fonctionner certaines imprimantes multifonctions ont besoin de paquets supplémentaires :

  • Imprimante HP, ajouter, si besoin, le paquet : hplip
  • Imprimante Canon, ajouter, si besoin, le paquet : cups-backend-bjnp ou télécharger les pilotes sur le site Canon.
  • Imprimante Brother, ajouter, si besoin, le paquet : printer-driver-brlaser
  • Imprimante Epson, si besoin, télécharger les pilotes sur le site Epson.

Dans ce mémo seule l'interface graphique "system-config-printer" sera détaillée.

Détection automatique de l'imprimante

  • Connecter l'imprimante au PC via un câble usb et éteindre l'imprimante.
  • Allumer le PC et rester sur la page du bureau Xfce

  • Allumer l'imprimante la détection automatique commence avec une notifiaction :
    Configuration de la nouvelle imprimante : Merci de patienter

  • Une nouvelle notification apparaît :
    L'imprimante a été ajoutée.

  • A la place de Description, se trouve le nom de l'imprimante qui a été ajoutée.

Utilisation de l'interface graphique "system-config-printer"

  • Ouvrir l'interface graphique : Applications > Paramètres > Configuration de l'impression (1 clic gauche)

  • L'application de l'interface graphique s'ouvre.

      1. Barre de menus : Serveur, Imprimante, Voir, Aide et le bouton Déverrouiller - Verrouiller
      2. Liste des imprimantes :
        • L'imprimante "Description" qui a été ajoutée.
        • PDF : Outil pour imprimer en pdf via le paquet : printer-driver-cups-pdf
      3. Barre d'état

  • Pour modifier les réglages des imprimantes, il faut déverrouiller la boite de dialogue.
    Placer le pointeur de la souris sur Déverrouiller et clic gauche

  • Une fenêtre apparaît, demandant le mot de passe pour Root (superutilisateur)

  • Saisir le mot de passe pour Root (superutilisateur) et valider avec Entrée ou cliquer > S'authentifier

  • Les modifications sont possibles puisque la boite de dialogue est déverrouillée.

  • Placer le pointeur de la souris sur l'imprimante "Description" et clic droit
    Une fenêtre apparaît :
    • Propriétés : Faire des modifications, imprimer une page de test, etc.
    • Supprimer l'imprimante
    • Activer ou désactiver l'imprimante
    • Définir par défaut
    • Voir la file d'attente des impressions

  • Image ci-dessus :
    "Définir par défaut" est grisé. Cela veut dire que l'imprimante est définie par défaut.
    Le rond vert avec le V blanc sur l'icône de l'imprimante indique que l'imprimante est définie par défaut.

  • Quitter l'application graphique : Serveur > Quitter

Paramètres du serveur (pour l'imprimante)

  • Placer le pointeur de la souris sur : Serveur > paramètres (clic gauche)

  • Une fenêtre apparaît, demandant le mot de passe pour Root (superutilisateur)

  • Saisir le mot de passe pour Root (superutilisateur) et valider avec Entrée ou cliquer > S'authentifier

  • Une fenêtre apparaît
    Placer le pointeur de la souris sur "Paramètres avancés du serveur", comme sur l'image ci-dessous
    Clic gauche, la page complète apparaîtra.

  • Configurer les paramètres du serveur comme sur l'image ci-dessous.

  • Nota : Résoudre des problèmes d'impression, l'image ci-dessus, cliquer sur "Problèmes ?" en haut à droite en bleu et suivre les instructions.
    Ou sur la barre de menus, cliquer sur Aide > Dépannage et suivre les instructions.

  • Quitter les paramètres du serveur : Cliquer > ok

  • Quitter l'application graphique : Serveur > Quitter

Ajouter (installer) manuellement une imprimante

  • Connecter l'imprimante au PC et l'allumer.
  • Avant toute modification, il faut déverrouiller la boite de dialogue, voir ci-dessus.
  • Placer le pointeur de la souris sur : Serveur > nouveau > imprimante (clic gauche)

  • Une fenêtre apparaît, choisisser le périphérique

  • Dans cette exemple, le choix est port série #1 (image ci-dessus)
    Si l'imprimante est connectée avec un câble usb, il faudra choisir :
    Sous, Périphérique : usb
    Sous, Connexion : usb
  • Puis, Cliquer > Suivant
    Attendre, le système recherche des pilotes.

  • Une fenêtre apparaît, choisisser l'imprimante
    Dans cette exemple : HP Deskjet 2721e
    Dans un premier temps sélectionner le fabricant ici : HP
    Faire défiler la liste soit avec la molette de la souris ou en plaçant le pointeur de la souris comme sur l'image ci-dessous et faire glisser la souris.
  • Cliquer > Suivant

  • Choisir le modèle de l'imprimante :
    Pour l'imprimante HP Deskjet 2721e, le modèle choisi est : Deskjet 2700
    Ne chercher pas "2721e": Dans cette exemple, il faut choisir la série "2700"
    Cliquer > Suivant

  • Une fenêtre apparaît avec la nouvelle description de l'imprimante
    Changer la description si besoin
    Cliquer > Appliquer

  • Une fenêtre peut apparaître, demandant le mot de passe pour Root (superutilisateur)
    Si non, passer l'étape

  • Saisir le mot de passe pour Root (superutilisateur) et valider avec Entrée ou cliquer > S'authentifier

  • Une fenêtre apparaît : Voulez-vous imprimer une page d'essai?
    Faire un choix, ici : Annuler
    Nota, l'imprimante doit être allumée pour imprimer une page de test...

  • L'imprimante HP Deskjet 2721e est sous le nom HP-Deskjet-2700
    De plus l'imprimante est définie par défaut (rond vert avec le V blanc)

  • Quitter l'application graphique : Serveur > Quitter
