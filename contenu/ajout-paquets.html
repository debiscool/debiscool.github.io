Ajout de paquets avec le Gestionnaire de paquets Synaptic.

Les applications sont organisées sous forme de paquets.
Le gestionnaire de paquets permet d'installer, de mettre à jour ou de supprimer ces applications.

ATTENTION :

  • Synaptic gère tous les programmes installés sur le système (PC).
  • Synaptic est à utiliser avec le plus grand soin.
  • Synaptic peut rendre le système inutilisable.

    1. Aller dans le menu : Applications > Système > Gestionnaire de paquets Synaptic

    2. Une fenêtre apparait, authentification requise avec le mot de passe pour Root (superutilisateur)

    3. Saisir le mot de passe pour Root (administrateur) et valider avec Entrée ou cliquer > S'authentifier

    4. La fenêtre de Synaptic apparaît
      Avant d'installer un paquet faire la mise à jour. Si la mise à jour a été éffectuée passer à l'étape suivante.

    5. Cliquer en haut à droite > Rechercher

    6. Une fenêtre apparaît, indiquer le paquet à installer exemple : komi, cliquer > Rechercher

    7. En bas à droite une barre d'état montre l'avancement de la recherche, patienter.

    8. La recherche terminée :
      En 1 : A gauche de l'écran en dessous de Toutes, s'affiche le nom du paquet qui a été saisi dans la fenêtre de recherche. Pour cet exemple, le paquet : komi
      En 2 : Le paquet komi apparaît dans "la liste des paquets" au milieu de l'écran.
      Cliquer > dans le petit carré blanc à côté de komi

    9. En 1 : Une fenêtre apparaît pour faire un choix : Sélectionner pour installation
      En 2 : La description du paquet sélectionné
      Cliquer dans la fenêtre > Sélectionner pour installation

    10. Pour fonctionner correctement, un paquet peut avoir besoin d'autres paquets appelés les dépendances.
      Une fenêtre peut apparaître pour ajouter une liste de paquets (les dépendances)
      Cliquer dans cette fenêtre > Ajouter la liste à la sélection

    11. Les paquets sélectionnés pour l'installation ont une flêche dans le petit carré.

      Pour information :
      En plus de la flêche dans le petit carré, les paquets qui ont été ajoutés à la sélection sont surlignés en vert.
      Dans cet exemple, les dépendances ne s'affichent pas sur la même page.
      Cliquer en haut à droite sur Rechercher, puis saisir la dépendance libmikmod3 > Rechercher

      En haut, le bouton "Appliquer" doit être en noir
      Cliquer sur > Appliquer, pour appliquer tous les changements à la selection.

    12. Après avoir cliqué sur Appliquer, une fenêtre apparaît indiquant :
      C'est votre dernière chance de parcourir la liste des chagements prévus avant qu'ils ne soient appliqués.
      Cliquer sur > Appliquer

    13. Des fenêtres apparaîssent indiquant l'avancement du téléchargement en cours, patienter.

    14. Un message informe que les modifications ont été appliquées, cliquer sur > Fermer

    15. Le petit carré à coté de "komi" doit être vert, idem pour ses dépendances.

    16. Pour avoir des informations sur le paquet qui vient d'être installé :
      Cliquer droit sur le nom du paquet, ici "komi"

    17. Cliquer sur > Propriétés, une fenêtre affiche toutes les informations du paquet.

    18. Fermer la fenêtre
    19. Fermer l'application Synaptic : fichier > Quitter

Après une installation, l'application doit se retrouver dans la liste des applications, classée dans un des thèmes.
Mais il y des cas exceptionnels comme "komi"

Donc, pour utiliser l'application qui vient d'être installée dans cet exemple :
Aller dans le menu : Applications > Exécuter le programme > saisir : komi > valider avec la touche Entrée du clavier.

Dans Gestionnaire de paquets Synaptic, cliquer sur Aide pour consulter la documentation.
Visiter la page d'accueil de Synaptic.
Consulter aussi : debian-facile pdf page 188 à 193, chapitre 8-5-4
