Debian avec le bureau Xfce

Avant de commencer, le PC doit être raccordé à internet. Ceci va permettre de télécharger un maximum de paquets lors de l'installation.

  • Installation stable de Debian GNU/Linux 11 (bullseye) avec le bureau XFCE.

1 - Installation stable de Debian GNU/Linux 11 (bullseye) avec le bureau XFCE.

Pour l'installation se référer aux liens ci-dessous.

  • Le site officiel : debian
  • Des sites en complément :
    • debian-facile pdf, page de 79 à 105, chapitre 5-4
    • l’administrateur Debian, chapitre 4-2
    • lecrabeinfo
    • malekal

Lors de l'installation cocher les cases comme ci-dessous.

Une fois l'installation terminée redémarrer le PC et laisser faire.
Le bureau Xfce va apparaître.

  • Indiquer :
    • L'identifiant utilisateur.
    • Le mot de passe utilisateur.
    • Valider avec Entrée.

2 - Configurer les sources de Apt.

Changer le fichier sources.list, qui se trouve dans : /etc/apt/sources.list

    1. Pour ouvrir le terminal aller dans : Applications - Emulateur de terminal.
    2. Pour se connecter en mode Root taper : su - (valider avec Entrée).
    3. Taper le mot de passe du mode Root, rien ne va apparaître à l'écran, valider avec Entrée.
    4. Si le caractère # s'affiche, alors mode Root
    5. Taper : nano /etc/apt/sources.list (valider avec Entrée).
    6. Remplacer l'ensemble par le fichier sources.list ci-dessous :

      7. Fichier sources.list wiki.debian

      Remplacer http parhttps :// ... dans toutes les lignes ci-dessus pour utiliser les dépots sur des connexions HTTPS chiffrées.

      Nota : Besoin de paquets non-free pour utiliser le wifi et le bluetooth.

    7. Enregistrer avec les touches du clavier : Ctrl et la lettre o (valider avec Entrée)
    8. Sortir avec les touches du clavier : Ctrl et la lettre x
    9. Retourner dans nano /etc/apt/sources.list, les modifications ont été enregistrées.
    10. Sortir avec les touches du clavier : Ctrl et la lettre x
    11. Sortir du mode Root taper : exit (valider avec Entrée)
    12. Sortir du mode simple utilisateur taper : exit (valider avec Entrée)

3 - Mise à jour en utilisant le Gestionnaire de paquets Synaptic.

  • Mise à jour : debian-facile PDF page 183 à 193, chapitre 8-5-1
    1. Aller dans : Applications - Système - Gestionnaire de paquets Synaptic
    2. Une fenêtre apparait, Taper le mot de passe Root
    3. Valider avec Entrée
    4. Cliquer en haut à gauche sur : Recharger et attendre la fin du chargement.
    5. Cliquer en haut à gauche sur : Tout mettre à niveau
    6. Une fenêtre apparaît avec la liste des paquets qui vont être mis à jour et les dépendances additionnelles si nécessaires.
    7. Ciquer dans la fenêtre : Ajouter à la selection.
    8. a) En haut à gauche "Appliquer" doit être en noir et non grisé. Cliquer sur Appliquer.
      b) Si "Appliquer" est grisé, ne rien faire le système est à jour. Passer à l'étape 12.
    9. Une fenêtre apparaît avec la liste des paquets
    10. Cliquer dans la fenêtre : Appliquer.
    11. Un message informe que les modifications ont été appliquées, Fermer la fenêtre
    12. Fermer l'application Synaptic : fichier - Quitter

4 - Ajouter des paquets en utilisant le Gestionnaire de paquets Synaptic.

  • Ajout de paquets : debian-facile PDF page 188 à 193, chapitre 8-5-4
    1. Aller dans : Applications - Système - Gestionnaire de paquets Synaptic
    2. Une fenêtre apparait, Taper le mot de passe du mode Root
    3. Valider avec Entrée
    4. Cliquer en haut à droite sur : Rechercher
    5. Une fenêtre apparaît indiquer le paquet à installer exemple : mousepad cliquer Rechercher
    6. Le paquet "mousepad" apparait, cliquer sur le carré à côté de "mousepad"
    7. Ciquer dans la fenêtre : Selectionner pour installation
    8. Si, Une fenêtre apparaît avec la liste des paquets
    9. Alors, cliquer dans la fenêtre : Ajouter la liste à la selection
    10. En haut à gauche "Appliquer" doit être en noir, cliquer sur Appliquer
    11. Un message informe que les modifications ont été appliquées, Fermer la fenêtre
      Le carré a coté de "mousepad" doit être vert.
    12. Fermer l'application Synaptic : fichier - Quitter
