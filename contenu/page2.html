Debian avec le bureau Xfce

4 - 3 - Ajout des paquets.

En plus des paquets qui ont été ajoutés lors de l'installation, installer la liste des paquets ci-dessous.
Cette liste n'est pas exhaustive et ce n'est pas le but. Elle permet d'avoir des outils supplémentaires.

  • Multimédia :
    • parole (déjà installé) : Lecteur de média pour l’environnement de bureau Xfce, ajouter :
      • gvfs
      • gvfs-backends
      • libdvdnav4
    • clementine : lecteur musical multiplateforme, permet aussi d'utiliser le lecteur CD
    • pulseaudio (déjà installé) : Contôle de volume
    • audacity : éditeur audio multi-piste, enregistre et lit des fichiers audio
    • xfburn (déjà installé) : graveur de cd et dvd par défaut
    • vlc : lecteur multimédia, à utiliser aussi pour la lecture de cd et de dvd, ajouter :
      • dvgrab
    • xcfa : outil pour extraire le contenu de CD audio et convertir les fichiers audio, ajouter :
      • cdparanoia
      • cd-discid
      • xcfa-dbg
    • webcamoid : application de capture de webcam
  • Bureautique :
    • le pack libre office doit être déjà installé
    • pdfarranger : application pour modifier les fichiers pdf
    • printer-driver-cups-pdf : outil pour imprimer en pdf. La première impression en pdf va créer
      un fichier pdf dans le Gestionnaire de fichiers
    • xfce4-screenshooter : utilitaire pour réaliser des captures d'écran du bureau
  • Graphisme :
    • gimp : éditeur d'images perfectionné
    • karbon : application de dessin
    • evince : visualisateur simple de documents, exemple des pdf
    • ristretto : visualisateur d’images, exemple des photos
    • eog : programme de visualisation d'images, possibilité d'imprimer l'image, exemple la photo
  • Connecter une source externe :
    • shotwell : permet d'importer des photos, images et vidéos depuis un disque ou d'un appareil photo ou d'un smartphone
    • simple-scan : application pour scanner
    • cups (déjà installé) : le système commun d'impression
    • jmtpfs : permet de connecter un appareil android, exemple le smartphone
  • Pare feu :
    • ufw : programme pour la gestion du pare-feu, ajouter :
      • gufw : l'interface graphique de ufw
  • Internet :
    • firefox-esr (déjà installé) : navigateur web puissant
    • thunderbird : le courrier électronique (exemple : recevoir les Emails sur le PC), ajouter :
      • thunderbird-l10n-fr : paquet de langue française
  • Système :
    • gparted : gestionnaire graphique pour partitionner les périphériques, exemple : disque dur ou clé usb
    • hardinfo : application qui affiche des informations sur le matériel et le système d'exploitation
  • Wifi et Bluetooth :
    • wireless-tools : outils pour les extensions sans fil, exemple : clé wifi, ajouter :
      • libiw30
    • firmware-iwlwifi (non free) : pour carte wifi intégré, ajouter :
      • initramfs-tools
    • wireless-regdb : pour carte wifi
    • bluez : permet d'utiliser les périphériques Bluetooth, ajouter :
      • bluetooth
      • blueman : interface graphique de gestion du bluetooth
    • firmware-realtek (non free) : firmware pour wifi et bluetooth
  • Webcam intégré et Webcam usb :
    • v4l-utils : fournit les utilitaires
