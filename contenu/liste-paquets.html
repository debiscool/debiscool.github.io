Liste des paquets à ajouter.

En plus des paquets qui ont été ajoutés lors de l'installation, installer la liste des paquets ci-dessous.
Cette liste n'est pas exhaustive et ce n'est pas le but. Elle permet d'avoir des outils supplémentaires.
Chaque paquet doit être installé un par un, en suivant le mémo Ajout de paquets.

  • Multimédia :
    • parole (installé par défaut) : Lecteur de média pour l’environnement de bureau Xfce, ajouter :
      • gvfs
      • gvfs-backends
      • libdvdnav4
    • clementine : lecteur musical multiplateforme, permet aussi d'utiliser le lecteur CD
    • pulseaudio (installé par défaut) : Contôle de volume
    • xfburn (installé par défaut) : graveur de cd et dvd
    • vlc : lecteur multimédia, à utiliser aussi pour la lecture de cd et de dvd, puis ajouter :
      • dvgrab
    • xcfa : outil pour extraire le contenu de CD audio et convertir les fichiers audio, puis ajouter :
      • cdparanoia
      • cd-discid
      • xcfa-dbg
    • webcamoid : application de capture de webcam
  • Bureautique :
    • le pack libre office doit être déjà installé
    • pdfarranger : application pour modifier les fichiers pdf
    • printer-driver-cups-pdf : outil pour imprimer en pdf. La première impression en pdf va créer
      un fichier pdf dans le Gestionnaire de fichiers
  • Accessoires :
    • xfce4-screenshooter (Capture d'écran) : utilitaire pour réaliser des captures d'écran.
  • Graphisme :
    • gimp : éditeur d'images perfectionné
    • karbon : application de dessin
    • evince (Visionneur de documents) : visualisateur simple de documents, exemple des pdf
    • ristretto : visualisateur d’images, exemple des photos
    • eog (Visionneur d'images) : programme de visualisation d'images, possibilité d'imprimer l'image, exemple la photo
  • Connecter une source externe :
    • shotwell : permet d'importer des photos, images et vidéos depuis un disque dur ou un appareil photo ou un smartphone, etc.
    • simple-scan : application pour scanner
    • cups (installé par défaut) : le système commun d'impression
    • jmtpfs : permet de connecter un appareil android, exemple le smartphone
  • Pare feu :
    • ufw : programme pour la gestion du pare-feu, puis ajouter :
      • gufw : l'interface graphique de ufw
  • Internet :
    • firefox-esr (installé par défaut) : navigateur web puissant
    • thunderbird : application de messagerie (Email), puis ajouter :
      • thunderbird-l10n-fr : paquet de langue française
  • Système :
    • gparted : gestionnaire graphique pour partitionner les périphériques, exemple : disque dur ou clé usb
    • hardinfo : application qui affiche des informations sur le matériel et le système d'exploitation
  • Wifi et Bluetooth :
    • wireless-tools : outils pour les extensions sans fil (exemple : clé usb wifi), puis ajouter :
      • libiw30
    • firmware-iwlwifi (non free) : pour carte wifi intégrée, puis ajouter :
      • initramfs-tools
    • wireless-regdb : pour carte wifi
    • bluez : permet d'utiliser les périphériques Bluetooth, puis ajouter :
      • bluetooth
      • blueman : interface graphique de gestion du bluetooth
    • firmware-realtek (non free) : firmware pour wifi et bluetooth
  • Webcam intégré et Webcam usb :
    • v4l-utils : fournit les utilitaires
