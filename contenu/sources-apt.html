Configurer les sources de Apt (advanced package tool).

Changer le fichier sources.list, qui se trouve dans : /etc/apt/sources.list

Pour cela 2 solutions : Synaptic ou par le Terminal

A - Utiliser SYNAPTIC (Interface graphique du gestionnaire de paquets Debian)
Les applications sont organisées sous forme de paquets.
Le gestionnaire de paquets permet d'installer, de mettre à jour ou de supprimer ces applications.

ATTENTION :

  • Synaptic gère tous les programmes installés sur le système (PC).
  • Synaptic est à utiliser avec le plus grand soin.
  • Synaptic peut rendre le système inutilisable.

AVANT de poursuivre, consulter SourcesList du wiki.debian

    1. Aller dans le menu : Applications > Système > Gestionnaire de paquets Synaptic

    2. Une fenêtre apparait, Authentification requise avec le mot de passe pour Root (superutilisateur)

    3. Saisir le mot de passe pour Root (superutilisateur) et valider avec Entrée ou cliquer > S'authentifier

    4. A la première ouverture du Gestionnaire de paquets Synaptic, la fenêtre de présentation rapide apparaît.
      Lire et cliquer > Fermer (il se peut que cette fenêtre n'apparaisse pas.)

    5. L'application de Synaptic apparaît, composée de 6 zones :

        1. Barre de menus : Fichier, Edition, Paquets, Configuration, Aide.
        2. Barre d'outils : Permet les actions principales. Recharger, Tout mettre à niveau, etc...
        3. Sélecteur de catégorie : Affine la liste des paquets
        4. Liste des paquets : Répertorie les paquets connus. Peut être optimisé à l'aide de filtres.
        5. Champ de description : Affiche la description du paquet sélectionné.
        6. Barre d'état : Affiche des informations globales sur l'état de Synaptic.

    6. Cliquer sur le menu > Configuration, une fenêtre apparaît

    7. Cliquer sur > Dépôts, une fenêtre des Dépôts apparaît, l'agrandir avec le petit carré en haut à droite de cette fenêtre.

    8. Si l'installation de Debian a été faite via un support externe (exemple : clé usb) associé à une connexion internet :
      Le fichier sources.list doit ressembler à ci-dessous.

    9. Les prochaines mises à jour se feront par internet. Il n'y a plus besoin de pointer sur le cdrom.
      Décocher la case cdrom

    10. Ajouter les composants "contrib non-free" pour faire fonctionner certaines cartes wifi intégrées,
      clés usb wifi et le bluetooth
      Et changement de http > https pour utiliser les dépots sur des connexions HTTPS chiffrées. wiki.debian

      11. Modification de :

      deb http://deb.debian.org/debian/ bullseye main
      deb-src http://deb.debian.org/debian/ bullseye main

      deb http://security.debian.org/debian-security/ bullseye-security main contrib
      deb-src http://security.debian.org/debian-security/ bullseye-security main contrib

      deb http://deb.debian.org/debian/ bullseye-updates main contrib
      deb-src http://deb.debian.org/debian/ bullseye-updates main contrib

      par :

      deb https://deb.debian.org/debian/ bullseye main contrib non-free
      deb-src https://deb.debian.org/debian/ bullseye main contrib non-free

      deb https://security.debian.org/debian-security/ bullseye-security main contrib non-free
      deb-src https://security.debian.org/debian-security/ bullseye-security main contrib non-free

      deb https://deb.debian.org/debian/ bullseye-updates main contrib non-free
      deb-src https://deb.debian.org/debian/ bullseye-updates main contrib non-free

      Cliquer sur la ligne > deb http://deb.debian.org/debian/ bullseye main

    11. En bas de la fenêtre saisir dans :

      12. URI : https://deb.debian.org/debian/

      Catégories : main contrib non-free

      Faire la même manipulation pour les 5 autres types d'archives.

    12. Les modifications terminées, cliquer en bas à droite > Accepter

    13. Une fenêtre apparaît "Dépôts modifiés", cliquer > Recharger
      Et laisser faire

    14. A ce stade vérifier bien les modifications dans : Configuration > Dépôts
      Si, les modifications se sont bien passées, quitter Synaptic.
      Sinon, recommencer
    15. Quitter le Gestionnaire de paquets Synaptic: Fichier > Quitter

    16. Vérifier la liste des changements dans sources.list
      Aller dans : Application > Gestionnaire de fichiers > Système de fichiers > etc > apt
      Positionner le pointeur de la souris sur "sources.list" > clic droit
      Puis : Clic gauche > Ouvrir avec "Mousepad", Mousepad est un éditeur de texte.

    17. L'éditeur de texte Mousepad s'ouvre. On doit retrouver les changements effectués.
      Fermer Mousepad : Fichier > Quitter
      Fermer le Gestionnaire de fichiers : Fichier > Fermer la fenêtre

B - Utiliser le Terminal : AVANT de poursuivre, consulter SourcesList du wiki.debian

Attention : Le Terminal est à utiliser avec le plus grand soin.

    1. Pour ouvrir le Terminal aller dans : Applications > Emulateur de terminal.

    2. Le Terminal s'ouvre en simple utilisateur car il y a "$"

    3. Pour se connecter au compte Root (superutilisateur) saisir : su - (valider avec la touche Entrée du clavier).

    4. Saisir le mot de passe du "superutilisateur" Root, rien ne va apparaître à l'écran (valider avec la touche Entrée du clavier).

    5. Si le caractère "#" s'affiche, alors bienvenue dans le compte administrateur.

    6. Saisir : nano /etc/apt/sources.list (valider avec la touche Entrée du clavier).

    7. Modification du fichier sources.list ci-dessous : wiki.debian

      8. Modification de :

      deb http://deb.debian.org/debian/ bullseye main
      deb-src http://deb.debian.org/debian/ bullseye main

      deb http://security.debian.org/debian-security/ bullseye-security main contrib
      deb-src http://security.debian.org/debian-security/ bullseye-security main contrib

      deb http://deb.debian.org/debian/ bullseye-updates main contrib
      deb-src http://deb.debian.org/debian/ bullseye-updates main contrib

      par :

      deb https://deb.debian.org/debian/ bullseye main contrib non-free
      deb-src https://deb.debian.org/debian/ bullseye main contrib non-free

      deb https://security.debian.org/debian-security/ bullseye-security main contrib non-free
      deb-src https://security.debian.org/debian-security/ bullseye-security main contrib non-free

      deb https://deb.debian.org/debian/ bullseye-updates main contrib non-free
      deb-src https://deb.debian.org/debian/ bullseye-updates main contrib non-free

      Nota : Besoin de paquets non-free pour utiliser certaines cartes wifi, clés usb wifi et le bluetooth.

    8. Enregistrer avec les touches du clavier : Ctrl et la lettre o (valider avec la touche Entrée du clavier)
    9. Sortir avec les touches du clavier : Ctrl et la lettre x
    10. Retourner dans nano /etc/apt/sources.list, les modifications ont été enregistrées.
    11. Sortir avec les touches du clavier : Ctrl et la lettre x

    12. Pour sortir du compte administrateur saisir : exit (valider avec la touche Entrée du clavier)

    13. On se retrouve sur le compte simple utilisateur

    14. Pour sortir du compte simple utilisateur saisir : exit (valider avec la touche Entrée du clavier)

Pour consulter la documentation du Terminal cliquer sur Aide.

Pour consulter la documentation de Synaptic, cliquer sur Aide.
Visiter la page d'accueil de Synaptic.
Consulter aussi : debian-facile pdf page 183 à 193, chapitre 8-5-1
